// Core Protocols

Trust & Sovereignty Architecture

PERMITTL Ring

The Master Key

Offline cold key generates scoped, time-limited worker keys. The master identity never touches the network. Three rings: Cold (USB+TPM), Semi-Cold (Vault), Hot (RAM/ephemeral).

Janus Protocol

Two-Object Requirement

Physical USB (Key of Janus) + logical software layer (Ring of Janus). Detached LUKS header — without the USB, the drive is indistinguishable from noise. JANUS VOID = 8-step wipe.

Golden Ticket (ZKP)

zk-SNARK Master Proof

gnark PLONK circuits. Client computes, DivaChain witnesses. Proves valid state transitions without revealing private keys or amounts. Bundles thousands of actions into one tiny proof.

Personal Chain

Private Local Ledger

Off-chain PLONK notebook tracking all activity before global submission. Bundles into Golden Tickets. Scales to laptop-grade hardware — the Screaming Demon baseline.

Ghost Gate

Witness Engine

Kernel-level nftables, Default DROP. Every packet fingerprinted to a signed identity. Cerberus 3-head authentication. Phantom chaff obfuscation. Zero implicit trust.

Magic Packet

Proof of Storage

Probabilistic storage verification. Random challenge packets prove a node holds data without revealing content. DivaChain's CAP-aware consensus layer — I2P partition-tolerant.
